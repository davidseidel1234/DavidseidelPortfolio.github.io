Final Content

For my final project in this class I decided to make another game in scratch. I improved my coding from my last game by including moving platforms, using blocks, and lists. I used blocks to make my code less lengthy and used a list so the player can see their scores at the end of the game. The game is called Electric man and the player moves the character with arrow keys. If the player touches the blue the player dies and has to restart the level. I included a death counter, timer, and level counter so the player can keep track of their score.
